📚 Leçons Disponibles

Leçon 1 : Les Équations du 1er degré

Définition : Une équation du 1er degré est de la forme ax + b = 0

📝 Exemple :

Résous : 2x + 4 = 10

Solution : 2x = 6 → x = 3 ✅

Leçon 2 : Les Fractions

Règle : Pour additionner 2 fractions, on met au même dénominateur.

📝 Exemple :

Calcule : 1/2 + 1/4

Solution : 2/4 + 1/4 = 3/4 ✅

Leçon 3 : La Puissance

Règle : a^n × a^m = a^(n+m)

📝 Exemple :

Calcule : 2^3 × 2^2

Solution : 2^(3+2) = 2^5 = 32 ✅
